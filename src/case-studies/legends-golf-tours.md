    body: >
      Each tour has a daily itinerary, with an animated progress bar to keep
      users scrolling down to the booking section.


      Navigation links are hidden when scrolling down, and show when scrolling up, so users only see content relevant to them in that moment.


      The navbar has a dynamic CTA button which is relevant to the current page, e.g. 'Book this tour' on tour pages.

      Using Webflow's CMS allowed the client to update information on a daily
